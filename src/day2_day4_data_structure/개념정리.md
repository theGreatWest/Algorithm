## 💻 배열

### 장점

- 인덱스 사용해 값에 바로 접근 가능
- 구조가 간단

### 단점

- 삽입/삭제 어려움
- 한 번 선언하면 크기를 늘리거나 줄일 수 없음

## 💻 리스트

### 장점

- 삽입/삭제하는 연산 속도 빠름
- 가변 데이터를 다룰 때 적절

### 단점

- 접근 속도 느림( 인덱스가 없어 값에 접근하려면 Head 포인트부터 순서대로 접근해야 함 )
- 구조 복잡

## 💻 구간 합

- 시간 복잡도를 줄이기 위해 사용하는 특수한 목적의 알고리즘
- 사용 빈도가 높음
- 합배열 **S[i] = S[i-1] + Arr[i]**
- **구간 합(startIndex, endIndex) = S[endIdx] - S[startIdx -1]**
